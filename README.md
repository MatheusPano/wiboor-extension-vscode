# Wiboor — extensão VSCode

Visualize e gerencie tarefas do Wiboor sem sair do vscode: uma aba dedicada na
Activity Bar lista suas tarefas; ao clicar, abre o detalhe com ações de **iniciar**, **pausar** e **finalizar**.


## Configuração

A primeira vez que você abre a aba da Wiboor, a extensão pede a **API key** com essa informação ela já pega o executorID e com isso ja estará pronto pra ser utilizada
